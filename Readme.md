# Azure Integration – Basic POC

## Overview
This project is a basic Proof of Concept (POC) created to understand Azure Integration fundamentals for a cloud-based application.

## Tech Stack
- Python
- FastAPI
- Uvicorn
- VS Code

## Functionality
- Simple backend application with REST endpoints
- Health check endpoint for monitoring
- Environment-based configuration to simulate cloud deployment behavior

## Azure Integration Concept
The application is designed to be cloud-ready and can be deployed on Azure App Service.
It demonstrates:
- Application hosting concept
- Environment configuration
- Basic monitoring via health endpoint

## Deployment Flow (Conceptual)
Local Development → Azure App Service → Application Monitoring

## Learning Outcome
- Understanding of Azure Integration fundamentals
- Cloud deployment awareness
- Documentation and reporting practice
